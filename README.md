# ezcut-gallery

[ezcut-gif](https://github.com/S-P-A-N/ezcut-gif)를 통해 잘라낸 애니메이션 타일 이모지들을 아카이빙하고 검색할 수 있는 정적 웹 갤러리입니다.

## 링크

- **갤러리 뷰어 (GitHub Pages)**: [https://S-P-A-N.github.io/ezcut-gallery/](https://S-P-A-N.github.io/ezcut-gallery/)
- **소스 저장소**: [https://github.com/S-P-A-N/ezcut-gallery](https://github.com/S-P-A-N/ezcut-gallery)

## 사용 안내

본 레포지토리는 직접 수정하기보다는 `ezcut-gif` 도구의 갤러리 업로드(Share) 기능을 통해 자동으로 데이터가 업데이트되도록 설계되었습니다. `emojis` 폴더 내부에 이미지 파일과 텍스트가 쌓이고, `gallery.json`이 갱신되면 GitHub Pages를 통해 웹사이트에 즉시 반영됩니다.
